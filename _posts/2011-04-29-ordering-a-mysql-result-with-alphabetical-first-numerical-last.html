Ordering a result by column in MySql does so alphanumerically. Items beginning with numerical characters will come first. Sometimes we want those entries starting with numbers to appear at the end.

This is how I did just that:

SELECT mycolumn FROM mytable ORDER BY CASE WHEN conv(mycolumn,10,10) = 0 THEN mycolumn ELSE concat('zzz',mycolumn) END

When the column's data fails a conversion to Base 10 it returns a zero and so does NOT contain exclusive numerical characters. If the conversion succeeds we know it's numerical and so it be can prefixed with a "zzz" and so listed at the end of the alphabetical results.

so

2 235 34 45.7 566 Alphaville Breathless

becomes

Alphaville Breathless 2 235 34 45.7 566
