If you've upgraded to 1.3.3 from an earlier release of Magento AND you have a custom CMS layout anywhere AND you experience this error:

Invalid id or tag ‘’ : must use only [a-zA-Z0-9_]

...you probably need to update the config of your module that creates the CMS layout config to include a <layout_handle&gt: node. See this reply/thread on the Magento forums:

http://www.magentocommerce.com/boards/viewreply/151194/
