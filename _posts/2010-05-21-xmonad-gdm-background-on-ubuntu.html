I'm enjoying using xmonad as a window manager on my linux box.

...but was bored of the default ubuntu background.

It can be changed by resetting the GDM background picture with GNOME's gconftool, like so:

sudo -u gdm gconftool-2 --set --type string \
--set /desktop/gnome/background/picture_filename \
/usr/share/backgrounds/cosmos/cloud.jpg

...which gives you a trippy cosmos cloud to stare at.
